## ✨ About This Project

Tahira's Kitchen is a clean, elegant single-page cooking website built with pure HTML, CSS, and JavaScript. It features a stunning gallery of dishes with ingredient highlights and direct links to YouTube videos showing the full cooking process.

Perfect for food creators who want to present their work beautifully while driving traffic to their YouTube channel.

## 🌟 Features

- **Elegant Dark & Gold Design** with beautiful typography
- **Responsive Gallery** with hover effects and smooth animations
- **Dish Cards** showing title, category, description & key visual appeal
- **Direct YouTube Integration** — click any dish to watch the video
- **Fully Responsive** — looks great on mobile, tablet, and desktop
- **Easy to Customize** via `dishes.json`
- **Contact Form** (ready for Formspree/Netlify Forms)
